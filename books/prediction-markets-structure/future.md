# 予測市場は公共インフラになれるか

予測市場の未来を考えるとき、最大の問いはこれです。

予測市場は、単なる投機サービスを超えて公共的な情報インフラになれるのか。

価格が未来に関する有力なシグナルになるなら、政策、企業意思決定、報道、リスク管理など様々な領域で使い道があります。特にAI時代には、この価値はさらに高まる可能性があると思っています。

# AIとの相性

AIは大量の情報を要約して、シナリオを生成して、更新頻度の高い状況認識を得意とします。

一方、予測市場はその情報に対して「どれくらい確信を持つか」を価格で表現するのが得意です。AIが情報を処理して、予測市場がその確度を評価する。この組み合わせは非常に相性が良いと思っています。

# 分岐する未来

予測市場の未来は一つじゃないと思います。

**規制金融市場として発展する道**  
Kalshi型のアプローチが広がって、event contractが金融商品として定着していく。

**オンチェーン情報市場として拡大する道**  
Polymarket型が世界中に広がって、ブロックチェーン上の情報インフラになっていく。

**企業内・限定環境で使われる道**  
社内予測市場として、組織の意思決定に組み込まれていく。

**日本のようにポイント型・参加型メディアとして再構成される道**  
金銭を直接賭けずに、予測市場の体験だけを借りる形で展開される。

どの道が正解かはわからないし、複数の道が並行して進むかもしれません。

# 最後に

予測市場は、未来を当てる魔法の装置じゃありません。

でも分散した情報を価格に変える制度として見たとき、その可能性は非常に大きいです。

問題は、予測市場が存在できるかどうかじゃありません。どのような設計なら社会的に受け入れられて、情報インフラとして機能するのか。その問いに対する答えは、まだ出ていません。

そしてまだ出ていないからこそ、いま予測市場を学ぶ意味があると思っています。

個人的には、この領域の動きを引き続きウォッチして、新しい展開があれば記事やコンテンツで共有していきたいと考えています。
