# 予測市場の定義

予測市場は、将来の出来事に関する予想を市場取引によって価格として表現する仕組みです。

もっと平たく言うと、「ある出来事が起こると思う人」と「起こらないと思う人」が売買を行って、その需給によって確率のように読める価格が形成される市場です。

![Polymarketのトップページ。地政学、政治、暗号通貨など様々なカテゴリの市場が並ぶ](/images/prediction-markets-structure/polymarket-top.jpg)
*Polymarketのトップページ（2026年3月時点）*

```mermaid
graph LR
    A[将来の出来事] --> B[YES / NO 契約を発行]
    B --> C[参加者が売買]
    C --> D[価格が形成される]
    D --> E[確率として解釈]
    E --> F[結果確定で決済]
```

代表的なのは二値型の市場です。たとえば「年内にアメリカで利下げがあるか」という問いに対して、YES と NO の契約が用意されます。YES が 0.68 ドルで取引されていれば、市場はその出来事の発生確率をおおむね 68% と見積もっていると読むことができます。

もちろん、この価格は自然法則のような真の確率じゃありません。あくまで、その時点で市場参加者が持っている情報、期待、恐れ、ポジション調整の結果として現れた価格です。それでも予測市場が注目されるのは、この価格がアンケートや評論よりも鋭い情報を含むことがあるからなんです。

# 賭博との違い

予測市場はよく賭博と混同されます。これは半分正しくて、半分誤解なんですよね。

確かに、将来の出来事に対してお金を賭けるという意味では賭博に近い側面があります。ただ予測市場がしばしば特別な扱いを受けるのは、単なる娯楽じゃなくて、情報集約機能を持つと考えられているからです。

たとえばスポーツベッティングは、基本的には娯楽としてのベット体験が中心です。一方で予測市場は、選挙、マクロ経済、政策、地政学など「社会的に情報価値のある未来予測」を価格として表現することに重きが置かれます。

もちろん、この違いは理論上のものであって、実務上はかなり曖昧です。特にスポーツや政治を扱う市場では、娯楽性と情報性は簡単に混ざります。そのため予測市場は常に「これは金融商品なのか、賭博なのか、情報市場なのか」という問いにさらされているわけです。

# 世論調査との違い

予測市場を理解するうえで、世論調査との違いは重要だと思います。

世論調査は「あなたは誰を支持しますか」「あなたはどう思いますか」と尋ねます。これは現在の意見を測る仕組みですよね。

一方、予測市場は「実際に何が起こると思うか」を問います。しかもその回答には金銭的インセンティブが伴う。自分はA候補を支持していても、B候補が勝つと思えば B に賭けることができます。ここでは願望ではなく期待値が重視されるんです。

この違いはかなり大きいです。世論調査は支持の分布を知るのには向いていますが、結果予測としては回答者の願望や社会的望ましさバイアスが入りやすい。一方、予測市場は資金を伴うため、より本音に近いシグナルを引き出しやすいと考えられています。

# 金融市場との近さ

予測市場は一見すると特殊なサービスに見えますが、構造としては金融市場にかなり近いです。

将来のイベントに対して契約が発行されて、その契約が売買されて、満期時の結果に応じて決済される。この点では先物やオプションなどのデリバティブと発想が重なります。

:::message
デリバティブ（派生商品）とは、別の資産やイベントの価格に連動する金融商品の総称です。

先物は「将来の価格で売買する約束」、オプションは「将来の価格で売買する権利」。

予測市場も将来の出来事に値段をつけている点で同じ構造ですが、「価格」ではなく「出来事が起きるかどうか」に連動するのが違いです。
:::

実際、Kalshi のようなサービスはイベント契約を商品先物規制の文脈で扱っています。
https://help.kalshi.com/en/articles/13823765-how-is-kalshi-regulated
つまり予測市場は「未来について賭ける場」であると同時に、「イベントに連動するデリバティブ市場」としても理解できるわけです。
