# 集合知という考え方

予測市場が注目される最大の理由は、「多数の人の情報を集約すると、個人よりも良い予測ができることがある」という集合知の発想にあります。

ある出来事について、人々はそれぞれ異なる断片情報を持っているんですよね。ニュースを追っている人、現場感覚を持っている人、マクロ指標を見ている人、政治の文脈に詳しい人、単純に統計に強い人。その全員が同じ場所で売買をすると、個々の断片情報が価格に反映されて、結果として一人では出せない見立てが形成される。これが予測市場の基本的な考え方です。

ここで重要なのは、全員が賢い必要はないという点です。大事なのは多様な情報があって、しかも当てた人が報われる設計になっていることなんです。

# ハイエクの分散知

この考え方は、フリードリヒ・ハイエクの「知識は社会の中に分散して存在する」という議論と非常に相性が良いです。

ハイエクは、中央の計画者がすべての情報を把握して最適な意思決定を行うのは難しいと考えました。なぜなら経済に必要な知識は、局所的で断片的で、その場その場の状況に埋め込まれているからです。

価格システムは、この分散した知識をシンプルなシグナルに変換する仕組みとして理解されます。予測市場も同じように、未来に関する分散知を価格へ変換する装置として見ることができます。

つまり予測市場は「みんなの意見を平均する仕組み」じゃなくて、「分散した知識を取引によって圧縮する仕組み」だと理解した方が本質に近いと思います。

ハイエクの原論文「The Use of Knowledge in Society」
https://fee.org/articles/the-use-of-knowledge-in-society/
1945年の論文ですが、予測市場の理論的支柱として今も引用されます。

# 効率的市場仮説との関係

予測市場は、効率的市場仮説ともよく結びつけて語られます。

効率的市場仮説は大雑把に言えば「公開情報は価格に織り込まれる」という考え方です。完全に正しいかどうかは別として、少なくとも金融市場において価格が重要な情報集約メカニズムであることは広く認められています。

予測市場も同じ発想で理解できます。もし新しいニュースや非対称的な情報があれば、それを知った参加者が売買によって利益を取ろうとする。その行動によって価格が更新される。結果として価格は最新情報に敏感なシグナルになるわけです。

この意味で、予測市場の価格は単なる多数決じゃありません。新情報に対するインセンティブ付きの反応なんです。

# なぜアンケートより強いことがあるのか

予測市場がアンケートや専門家コメントより強いことがあるのは、主に3つの理由によります。

**本音が出やすい**  
アンケートは社会的望ましさバイアスが入りやすいですが、市場は金銭的インセンティブが伴うため本音が反映されやすい。

**情報更新が速い**  
新しいニュースが出たらすぐに価格が動きます。アンケートは再実施に時間がかかりますが、市場は24時間動いています。

**少数の正しい人が価格を動かせる**  
正しい情報を持っている少数の参加者が、大きなポジションを取ることで価格を動かせます。多数決では埋もれてしまう情報が、市場では反映されやすい。

この3点が、予測市場を単なる人気投票と区別します。

# それでも市場は万能ではない

ただし、市場が未来を当てるという語り方はかなり危険だと思っています。

市場価格は真理じゃありません。流動性が薄い、参加者が偏っている、ノイズトレーダーが多い、操作コストが低い、解決ルールが曖昧、といった条件では簡単に歪みます。

つまり予測市場が強いのは「市場だから」じゃないんです。適切な流動性があって、情報が分散していて、ルールが明確で、操作や内部情報への対処がある程度機能しているときに限って、価格は強いシグナルになります。

:::message alert
「予測市場の価格 = 真の確率」ではありません。流動性や参加者の偏りによって簡単に歪むため、価格の読み方には注意が必要です。
:::
