# Kalshiの意味

Kalshi は、予測市場を制度の外で拡張するのではなく、制度の中で正面から成立させようとした存在です。

ここが Polymarket と決定的に違います。Polymarket がコード、トークン、透明性、グローバル性から広がったのに対して、Kalshi は規制、許認可、監督、市場運営の正統性から広がりました。

# event contractとしての予測市場

Kalshi の文脈では、予測市場はしばしば event contract として理解されます。これは、特定の出来事が起きたかどうか、ある閾値を超えたかどうか、といったイベントに連動して価値が決まる契約です。

# 制度がもたらすもの

規制市場型の強みは、ルールと執行の明確さです。KYC、アクセス制御、市場監視、不公正取引への対応などが取り入れやすくなります。

# 規制型の限界

ただし、規制型だから無敵というわけではありません。扱えるテーマに制約がかかりやすく、政治、スポーツ、戦争、暗殺、地政学など、社会的にセンシティブな市場では金融商品として許容されるかがすぐに問題になります。

# Polymarketとの対比

Kalshi と Polymarket を比べると、両者は同じ予測市場でも土台がかなり違います。Kalshi は制度を信頼の源泉にし、Polymarket は透明性とオープンな構造を信頼の源泉にします。
